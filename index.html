Как это работает?

🚀

Push в репозиторий

Вы просто отправляете изменения в ветку main вашего Git-репозитория, как при обычной разработке.

⚙️

Запуск пайплайна

GitHub Actions перехватывает событие, собирает проект, оптимизирует код и готовит артефакты.

☁️

Публикация в Azure

Готовые файлы автоматически доставляются в Azure Static Web Apps и становятся доступны всему миру.
